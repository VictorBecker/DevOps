Este projeto foi desenvolvido em Java como parte de uma atividade acadêmica e está versionado utilizando o GitHub.
Além disso, foi utilizado o Docker para containerizar a aplicação, permitindo que ela seja executada de forma padronizada em diferentes ambientes, sem a necessidade de configurações adicionais na máquina do usuário.


Projeto da matéria Fundamentos da Programação Orientada a Objetos usado para alcancar os objetivos da Somativa 1 da matéria DevOps utilizando metódos de controle de versão
